# 서대문구 · 서울특별시

> 서대문구 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-seoul-seodaemun.github.io/**

서대문구는 독립문과 서대문형무소역사관으로 상징되는 항일 독립운동의 성지이자, 신촌 일대 대학가를 품은 서울의 대표적 교육·문화 중심지입니다. 구 이름은 한양 도성의 서쪽 대문인 돈의문(敦義門)에서 유래했으며, 정작 그 문터는 오늘날 종로구에 남아 있어 지명과 위치가 어긋나는 독특한 사연을 지녔습니다. 연세대학교와 이화여자대학교를 비롯한 다수의 대학이 밀집해 젊은 활력이 넘치고, 매년 여름 연세로에서 열리는 신촌물총축제는 도심 속 이색 여름 명물로 자리 잡았습니다. 안산자락길과 홍제천 인공폭포 같은 무장애 자연 명소가 도심 한복판에서 숲과 물길을 선사한다는 점도 이 구의 매력입니다. 독립운동 역사와 청년 문화, 그리고 접근성 높은 녹지가 한데 어우러진 곳이 바로 서대문구입니다.
